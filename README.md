- **Database:** MySQL / MariaDB
- **Version Control:** Git & GitHub
- **Editor:** Visual Studio Code

---

## ⚙️ Local Setup Instructions

1. **Install XAMPP** from [https://www.apachefriends.org](https://www.apachefriends.org)
2. **Clone the repository** into your `htdocs` folder:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Apex_planet_project.git
   ```
3. **Start Apache & MySQL** via the XAMPP Control Panel
4. **Visit** `http://localhost/Apex_planet_project/` in your browser
